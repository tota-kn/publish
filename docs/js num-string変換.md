# num to string
```js
const num = 123;
String(num); // "123"

const flo = 123.45678;
String(flo); // "123.45678"
```

# string to num
```js
var str = '123';
Number(str); // 123.45678

var str = '123.45678';
Number(str); // 123.45678
```
