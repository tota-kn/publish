# 幅指定
`sips --resampleWidth <幅> <縮小したい画像パス> --out ./out.png`
# 高さ指定
`sips --resampleHeight <高さ> <縮小したい画像パス> --out ./out.png`
# フォーマット変換
`sips -s format <format> <originalfilepath> -o <changedfilepath>`
