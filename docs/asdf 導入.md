# ダウンロード
`brew install asdf`

# 初期設定
インストール方法と利用shelllによって方法が異なるので公式ページを参照
https://asdf-vm.com/guide/getting-started.html#_1-install-dependencies

fish+brewの場合は以下コマンド
```sh
echo -e "\nsource "(brew --prefix asdf)"/libexec/asdf.fish" >> ~/.config/fish/config.fish
```
