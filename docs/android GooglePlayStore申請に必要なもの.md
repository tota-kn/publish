# 必要なもの
## aabファイル

## アイコン
- PNG
- 512px * 512px 

## フィーチャーグラフィック
- PNG
- 1MB以下
- 1024px * 500px

## スクリーンショット
- PNG
- 8MB以下
- アスペクト比16:9
- 縦横ともにが320~3840px
- 携帯・タブレット用にそれぞれ2~8枚（携帯・タブレットは同じ画像でも良い）
