# js date型の加算
```js
var dt = new Date();

//３か月後
dt.setMonth(dt.getMonth() + 3);

//３か月前
dt.setMonth(dt.getMonth() - 3);

//3日後
dt.setDate(dt.getDate() + 3);

//3時間後
dt.setHours(dt.getHours() + 3);

//3分後
dt.setMinutes(dt.getMinutes() + 3);
```
