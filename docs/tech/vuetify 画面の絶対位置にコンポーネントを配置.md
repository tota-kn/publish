# vuetify 画面の絶対位置にコンポーネントを配置
`fixed` または `absolute` と `top bottom right left`を使う
```vue
<v-btn fixed bottom right fab color="secondary">  
 <v-icon> mdi-plus </v-icon>  
</v-btn>
```
