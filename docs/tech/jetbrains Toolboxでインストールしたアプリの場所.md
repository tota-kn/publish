# jetbrains Toolboxでインストールしたアプリの場所
[[about Toolbox]]でインストールしたアプリは通常とインストール先が異なる。

下記に配置される。
```
$HOME/Library/Application Support/JetBrains/Toolbox/apps/<アプリ名>/<ch>/<version>/
```

ToolboxのSettingsからも確認できる。
