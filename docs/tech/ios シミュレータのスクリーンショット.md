# ios シミュレータのスクリーンショット
シミュレータを起動した状態で以下コマンドを打つ

## 保存
Command+S

## 保存先を指定して保存
Command + Alt + S
指定先は以降デフォルトの保存先となる

## クリップボードにコピー
Command + Control + C
