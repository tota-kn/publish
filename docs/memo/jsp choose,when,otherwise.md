# jsp choose,when,otherwise
```html
<c:choose>  
	<c:when test="${age >= 30}">３０代以上</c:when>  
	<c:when test="${age >= 20}">２０代</c:when>  
	<c:when test="${age >= 10}">１０代</c:when>  
	<c:otherwise>１０歳以下</c:otherwise>  
</c:choose>
```
