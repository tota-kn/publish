# mac DualShockを接続してもSteamのゲームで反応しない
Steamクライアントにコントローラー入力が奪われているよう

1. Steamクライアントでコントローラを使いたいゲームの起動画面を開く
2. 歯車マーク > プロパティ をクリック
3.  コントローラ > オーバーライド で`Steam入力を無効にする`を選択する
