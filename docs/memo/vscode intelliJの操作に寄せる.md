# 設定
```
# ダブルクリックでタブを開くようにする
"workbench.editor.enablePreview": false
"workbench.editor.enablePreviewFromQuickOpen": false
"workbench.list.openMode": "doubleClick"

# エクスプローラービューが選択したタブのファイルに移動しないようにする
"explorer.autoReveal": false
```
