# vscode ショートカット
## メニューフォーカス
| 操作                     | ショートカットキー   |
| ------------------------ | -------------------- |
| エディター               | command + 1 ~ 8      |
| タブ移動                 | Ctrl + (shift) + tab |
| 表示しているサイドバー   | command + 0          |
| ファイルエクスプローラー | command + shift + E  |
| 全文検索                 | command + shift + F  |
| git                      | control + shitf + G  |
